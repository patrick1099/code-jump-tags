# Security

Please do not disclose suspected security issues in public comments or reviews.

If this project is hosted in a public GitHub repository, use the repository's private security advisory flow. If you received the extension through another channel, report the issue privately to the publisher or maintainer who provided that channel.

When reporting an issue, include:

- Extension version.
- VS Code version and operating system.
- Steps to reproduce.
- Impact and any proof-of-concept details you can safely share.

This project is a community-maintained fork and is not covered by Microsoft security response channels.
